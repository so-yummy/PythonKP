	Interf.py — графический интерфейс (GUI)

Создаёт окно с полями ввода, кнопками и радиокнопками.
Позволяет выбрать:
	Размер ключа (бит).
	Количество экспериментов.
	Тип простых чисел: обычные, близкие, сильные.

Есть кнопки:
	«Сгенерировать» — открыть окно для генерации простых чисел.
	«Графики» — показать графики по времени факторизации.
	«Запустить эксперимент» — генерирует ключи RSA и измеряет время взлома.

Эксперименты выполняются в отдельном потоке, чтобы окно не зависало.
Вывод результатов происходит в текстовое поле.


	Func.py — генерация простых чисел

Класс Generator содержит методы:
	MillerRabin(n, k) — проверка числа на простоту с помощью теста Миллера-Рабина.
	GeneratePrime(bit) — генерирует случайное простое число нужной битовой длины.
	GenerateStrongPrime(bit) — генерирует «сильное» простое число (p = 2*q1*q2 + 1).
Используется при генерации ключей RSA.


	RSA.py — генерация ключей

Класс RSA создаёт ключи: p, q и модуль n = p*q.
В конструкторе можно выбрать:
	st=True — использовать сильные простые числа.
	c=True — сделать p и q близкими для демонстрации уязвимости.
После генерации ключ готов для экспериментов.

	Factor.py — факторизация

Класс Factorization с методами:
	TrialDiv(n) — метод перебора, делит n на все числа до √n.
	Fermat(n) — метод Ферма, эффективен для близких простых p и q.
Используется для измерения «взлома» ключа в экспериментах.

	TimingAttack.py — демонстрация атаки по времени
Симулирует атаку по времени:
Уязвимая операция возведения в степень зависит от битов ключа. Можно измерить время и частично восстановить ключ.

	Exp.py — эксперимент

Класс Experiment проводит серию тестов:
	Генерирует несколько ключей RSA.
	Измеряет время факторизации каждого ключа.
	Сохраняет результаты в pandas DataFrame.

	Graph.py — визуализация
Функции для построения графиков времени факторизации:
	PlotRes(df) — среднее время в зависимости от типа ключей.
	PlotExperiments(df) — отдельные точки каждого эксперимента.
	PlotWithTheory(df) — сравнение эксперимента с теоретическим ростом сложности. (Он не работает)

То есть

1. GUI (Interf.py) собирает параметры от пользователя.
2. Генератор (Func.py) создаёт p и q.
3. Класс RSA (RSA.py) формирует ключ n = p*q.
4. Эксперимент (Exp.py) измеряет время взлома методом факторизации (Attack.py).
5. Результаты заносятся в DataFrame и выводятся в поле вывода.
6. Графики (Graph.py) строят зависимости времени от параметров ключа.

При выборе близких или сильных простых чисел изменяется сложность взлома (и время эксперимента).
